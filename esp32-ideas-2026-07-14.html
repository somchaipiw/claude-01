🤖 AI/ML 2026
LiteRT for Microcontrollers — TinyML ยุคใหม่บน ESP32-S3

Google รีแบรนด์ TensorFlow Lite Micro เป็น LiteRT for Microcontrollers ปี 2024 และในปี 2026 ได้เพิ่ม INT4 quantization (แม้ coverage ยังไม่ครบทุก operator) ทำให้ model ขนาดเล็กลงอีก

ESP32-S3 ที่มี vector instructions + PSRAM 8 MB รัน quantized CNN model ได้ใน หลักสิบ milliseconds กินไฟเฉลี่ยหลักเดียว mA

Quantization เปรียบเทียบ (2026)

INT8มาตรฐาน — model เล็ก 4× inference เร็ว 2-4× accuracy ลด 1-3%
INT4ใหม่ปี 2026 — model เล็กกว่า INT8 อีก 2× เหมาะ model ใหญ่
FLOAT32ช้า/กินแรม — หลีกเลี่ยงบน MCU

Thread Border Router ด้วย ESP32-C6 + ESP32-H21

Espressif รองรับ Thread 1.3 บน ESP32-C6 (Border Router) และ ESP32-H21 (End Device) ครบวงจรในระบบเดียว เชื่อมต่อ Zigbee/Thread devices เข้า Matter network ได้โดยตรง ไม่ต้องพึ่ง Hub จากยี่ห้ออื่น

  • 1ESP32-C6 ทำหน้าที่ Thread Border Router — เชื่อม Thread network กับ Wi-Fi/IPv6
  • 2ESP32-H21 เป็น End Device — sensor node กินไฟต่ำมาก (5µA deep sleep)
  • 3Devices แสดงผลเป็น Matter device ใน Home Assistant / Apple Home / Google Home
  • 4ไม่ต้องมีอินเทอร์เน็ตก็ทำงานได้ — local mesh network ล้วน

🌿 Air Quality
CO₂ Monitor แบบ Low-Power — SCD41 + ESP32-C3 + ePaper

SCD41 (Sensirion) ใช้เทคนิค NDIR วัด CO₂ จริง (ไม่ใช่ estimate) ความแม่นยำ ±40 ppm มี single-shot mode สำหรับ ultra-low-power — วัดแล้วหลับ ทำให้ใช้แบตเตอรี่ได้นานหลายเดือน

  • 1SCD41 ต่อ I2C กับ ESP32-C3 (3.3V, GPIO8=SDA, GPIO9=SCL)
  • 2ตั้งค่า single-shot measurement — ตื่นทุก 5 นาที วัด CO₂+Temp+RH แล้วนอน
  • 3แสดงผลบน ePaper display 2.13" — ไม่ต้องใช้ไฟตลอดเวลา
  • 4ส่งข้อมูล MQTT ไป Home Assistant ผ่าน ESPHome (ถ้า Wi-Fi เปิด)
📊 CO₂ ในห้อง <800 ppm = ดี | 800-1200 ppm = เริ่มง่วง | >1200 ppm = ต้องระบายอากาศ!

🌐 Mesh IoT
Thread Sensor Farm — ESP32-H21 Network แบตเตอรี่อยู่ปีๆ

สร้าง Sensor Farm ในสวน/โรงงาน ด้วย ESP32-H21 nodes เชื่อมกันเป็น Thread mesh ข้อมูลไหลผ่าน Border Router (ESP32-C6) เข้า Home Assistant โดยอัตโนมัติ ไม่ต้องเดินสาย

  • 1ESP32-H21 + SHT41 (Temp/RH) + CR2032 — ตรวจสอบดิน/อุณหภูมิทุก 10 นาที
  • 2Thread mesh กระจาย range — node A ถ่ายทอดข้อมูลให้ node ไกลๆ ได้
  • 3ESP32-C6 (Border Router) รับข้อมูลทั้งหมด ส่งต่อเข้า Home Assistant ผ่าน Wi-Fi
  • 4แบตเตอรี่ CR2032 อยู่ได้ 1-2+ ปี เพราะ deep sleep 5µA ระหว่าง interval
