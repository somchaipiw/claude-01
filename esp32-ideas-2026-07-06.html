Espressif ใหม่

ESP32-H21 — Ultra-Low-Power BLE 5 + 802.15.4 รุ่นใหม่

ESP32-H21 เป็นรุ่นต่อจาก H2 ที่ออกแบบมาเพื่อ smart home sensor โดยเฉพาะ ด้วย on-chip DC-DC converter ที่ช่วยลด active current ลงได้มากกว่าเดิม เหมาะสำหรับแบตเตอรี่อายุยาว

  • CPU32-bit RISC-V (single core)
  • WirelessBLE 5.3 + IEEE 802.15.4
  • ProtocolZigbee 3.0 + Thread 1.3 + Matter
  • พลังงานOn-chip DC-DC ลด active current
  • Use CaseBattery sensor, wearable, automation
สถาปัตยกรรม 2026: แนวทางที่ดีที่สุดคือ ESP32-C6 เป็น Border Router/Hub + ESP32-H21 เป็น end device ประหยัดพลังงาน ทำให้ Smart Home mesh ทำงานได้นานขึ้น
esp32-h21 zigbee 3.0 thread 1.3
ESP32-P4

ESP32-P4-EYE — บอร์ด Vision AI พร้อม MIPI Camera

Espressif เปิดตัว ESP32-P4-EYE development board สำหรับงาน Edge AI ด้าน Computer Vision โดยเฉพาะ มาพร้อม MIPI-CSI camera connector, MIPI-DSI display, และ AI instruction extensions บน dual-core RISC-V 400MHz

  • CPUDual-core RISC-V @ 400 MHz
  • PSRAMUp to 32 MB
  • CameraMIPI-CSI interface
  • DisplayMIPI-DSI interface
  • AIAI instruction extensions + FPU
ข้อสังเกต: ESP32-P4 ไม่มี Wi-Fi/BT ในตัว — ต้องจับคู่กับ ESP32-C6 หรือ S3 เพื่อ wireless connectivity ซึ่งทำให้ออกแบบ product ได้ยืดหยุ่นขึ้น
esp32-p4-eye edge-ai computer-vision
TinyML 2026

INT4 Quantization บน ESP32 — AI ขนาดเล็กลงอีก

ปี 2026 TinyML เข้าสู่ production จริงแล้ว โดย TensorFlow Lite Micro รองรับ INT4 quantization (บางส่วน) ทำให้ model size เล็กลง ~50% เมื่อเทียบกับ INT8 เหมาะสำหรับ ESP32-S3 ที่มี Vector Instructions

  • FrameworkTensorFlow Lite Micro
  • QuantizationINT4 (partial), INT8 (full)
  • ToolchainEdge Impulse (production-ready)
  • HW BestESP32-S3 (Vector) / P4 (AI ext)
  • Use CaseVoice, image classification, anomaly
สิ่งที่ต้องระวัง: INT4 ยัง partial operator coverage — ตรวจสอบ ops ที่ใช้ใน model ก่อน deploy หรือใช้ INT8 fallback สำหรับ production ที่ต้องการเสถียรภาพ
tinyml int4-quantization edge-impulse

Water Quality

เครื่องวิเคราะห์คุณภาพน้ำแม่น้ำด้วย ESP32

สร้างอุปกรณ์วัดคุณภาพน้ำแบบพกพา รวม pH, Conductivity และ Turbidity ในตัวเดียว ต้นทุนรวมประมาณ $50 USD ส่งข้อมูลขึ้น Cloud ผ่าน Wi-Fi แบบ real-time

  • ต่อ pH sensor (SEN0161) เข้า analog pin A0 ของ ESP32
  • ต่อ TDS sensor วัดค่า conductivity ผ่าน ADC
  • ใช้ turbidity sensor SEN0189 สำหรับความขุ่น
  • ส่งข้อมูลไปยัง Blynk หรือ ThingSpeak
  • เพิ่ม GPS module เพื่อ log ตำแหน่งที่เก็บตัวอย่าง

Smart Home

Zigbee Sensor Node ด้วย ESP32-H21 อายุแบตเตอรี่ 2+ ปี

สร้าง Zigbee end device ขนาดเล็กวัดอุณหภูมิ/ความชื้นด้วย ESP32-H21 จับคู่กับ Home Assistant ผ่าน ESP32-C6 border router เป้าหมายอายุแบตเตอรี่ CR2032 นานกว่า 2 ปี

  • ใช้ SHTC3 (I2C) วัด temp/humidity ความแม่นยำสูง
  • ตั้ง deep sleep interval 5 นาที ตื่นแค่ 200ms
  • Join Zigbee network ผ่าน H21 IEEE 802.15.4
  • C6 ทำหน้าที่ Thread/Zigbee border router
  • Home Assistant รับข้อมูลผ่าน Matter protocol
esp32-h21 zigbee 3.0 🔥 ยาก

ESP-NOW Mesh

เครือข่ายเซนเซอร์ไร้สาย ESP-NOW — ไม่ต้องพึ่ง Router

ESP-NOW เป็น protocol ที่ใช้ได้โดยตรง ไม่ต้องมี Wi-Fi router สร้าง mesh network ระยะสั้นระหว่าง ESP32 หลายตัว เหมาะสำหรับฟาร์ม, โรงงาน หรือพื้นที่ที่ไม่มี internet

ข้อดีหลัก: Latency ต่ำมาก (~1ms), ใช้พลังงานน้อย, ไม่มี DHCP overhead, รองรับ unicast/broadcast/multicast, pair สูงสุด 20 peers ต่อ ESP32 หนึ่งตัว
🌾 Smart Farm Sensor Grid

ESP32 หลายจุดส่งค่า soil moisture, อุณหภูมิ ไปยัง gateway ตัวเดียว

🏭 Industrial Monitor

วัด vibration, temperature ของ machine หลายจุด ส่งข้อมูลแบบ real-time

🚨 Emergency Alert

ระบบแจ้งเตือนไฟไหม้/น้ำท่วมแบบ offline ที่ทำงานได้แม้ไม่มี internet

🎮 Wireless Game Controller

สร้าง controller custom ด้วย ESP32 สื่อสาร ESP-NOW กับ receiver แบบ ultra-low latency

Water Quality

SEN0161 — Analog pH Sensor

วัดค่า pH ของน้ำในช่วง 0–14 ใช้ BNC connector เชื่อมกับ probe แยก เชื่อมต่อกับ ADC ของ ESP32 ได้โดยตรง

  • ช่วงวัดpH 0–14
  • อินเทอร์เฟซAnalog (0–3.3V)
  • ความแม่นยำ±0.1 pH
  • ราคาประมาณ$10–15 USD

Gravity TDS Sensor — วัด Total Dissolved Solids

วัดปริมาณสารละลายในน้ำ (TDS) บอกคุณภาพน้ำดื่ม/น้ำปลูกพืช hydroponics ราคาถูก เชื่อมต่อ analog กับ ESP32 ADC ได้ทันที

  • ช่วงวัด0–1000 ppm
  • อินเทอร์เฟซAnalog (0–2.3V)
  • การใช้งานHydroponics, drinking water
  • ราคาประมาณ$6–10 USD

Air Quality

SGP41 — VOC + NOx Sensor (Sensirion)

Sensirion SGP41 วัดทั้ง Volatile Organic Compounds และ Nitrogen Oxides ในตัวเดียว ใช้ I2C ต่อกับ ESP32 ตรง ความแม่นยำสูง พัฒนาร่วมกับ SCD41 เพื่อ full indoor AQ

  • วัดVOC Index + NOx Index
  • อินเทอร์เฟซI2C (0x59)
  • แรงดัน1.7–3.6V
  • Self-calibrationอัตโนมัติ

MAX30102 — Heart Rate + SpO2 Sensor

  • วัดHR + SpO2
  • อินเทอร์เฟซI2C (0x57)

SCD41 — CO2 Sensor (Photoacoustic)

LD2410 — mmWave Human Presence Radar

Radar คลื่น mmWave 24GHz ตรวจจับการมีอยู่ของคนแม้หยุดนิ่ง (presence detection) ไม่เหมือน PIR ที่ต้องการการเคลื่อนไหว เชื่อมต่อ UART กับ ESP32 ใช้ใน Home Assistant ได้ผ่าน ESPHome

  • ตรวจจับStationary + Moving presence
  • อินเทอร์เฟซUART
  • ราคาประมาณ$8–12 USD
