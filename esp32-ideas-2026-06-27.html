🌡️ Sensors ที่น่าสนใจ (ไม่ซ้ำกับฉบับก่อน)
HLK-LD2410 / LD2410B mmWave Radar

เซ็นเซอร์ radar 24GHz สำหรับ Human Presence Detection ที่ได้รับความนิยมอย่างมากในปี 2026 โดยเฉพาะกลุ่ม Home Assistant / ESPHome

  • ตรวจจับทั้งคนเคลื่อนไหว และคนนิ่ง (ตรวจ micro-movement จาก breathing)
  • ระยะ stationary 6m / moving 9m แบ่ง detection zone ได้
  • สื่อสารผ่าน UART 256000 baud + digital output pin
  • รองรับ ESPHome natively — ใช้กับ LD2410 component ได้เลย
  • ใช้งาน: smart lighting, ตรวจคนในห้อง, ระบบ security
NPK RS485 Soil Sensor Smart Agriculture

เซ็นเซอร์วัดธาตุอาหารหลักในดิน (Nitrogen, Phosphorus, Potassium) + อุณหภูมิ + ความชื้น + pH + EC แบบครบใน module เดียว

  • สื่อสาร RS485 Modbus — เชื่อมต่อกับ ESP32 ผ่าน MAX485 module
  • งานวิจัยล่าสุดใช้ Random Forest ทำนาย NPK ได้ accuracy 93.1%
  • ส่งข้อมูลผ่าน MQTT → Node-RED + Grafana dashboard
  • ต่อ relay ควบคุมปั๊มน้ำ/โซลินอยด์ valve ได้อัตโนมัติ
  • ใช้กับ ESP32 LoRa สำหรับพื้นที่ไม่มีสัญญาณ
Turbidity + Flow + ORP Water Quality IoT

ชุดเซ็นเซอร์วัดคุณภาพน้ำแบบครบระบบ งานวิจัย 2026 ใช้ TinyML บน ESP32 classify ประเภทน้ำได้ accuracy 99.28%

  • Turbidity sensor (nephelometric method) — วัดความขุ่น NTU
  • YF-S201 / YF-DN50 Flow Meter — วัดอัตราการไหล L/min
  • ORP sensor — วัดศักย์ redox บ่งบอกสถานะออกซิเดชัน
  • Dataset 6,000 points classify: Normal / Rainwater Runoff / Chemical
  • ส่งข้อมูลสู่ ThingSpeak — ดูผลจาก dashboard บน mobile ได้ real-time
MEMS Microphone Array Acoustic / AI

ใช้ microphone array INMP441 (I2S) หลายตัวร่วมกับ ESP32-P4/S3 สำหรับ anomaly detection เสียงเครื่องจักรในโรงงาน

  • INMP441 — MEMS I2S digital microphone ราคาถูก ใช้ได้กับทุกชิป ESP32
  • ESP32-P4 รวมกับ camera feed → multi-modal anomaly detection
  • ตรวจ CNC machine, packaging line, textile loom ก่อน failure
  • TFLite Micro + sound spectrogram → classify เสียงผิดปกติ
  • Wake-word detection เป็น classic TinyML demo บน ESP32-S3
Capacitive Soil Moisture Agriculture Sensor

เซ็นเซอร์ความชื้นดินแบบ capacitive (FC-28, v1.2) ไม่กัดกร่อน — แม่นยำกว่า resistive sensor รุ่นเก่า

  • ใช้หลักการเปลี่ยนความจุไฟฟ้าเมื่อดินมีความชื้น ทนทานกว่าแบบ resistive
  • เชื่อมต่อ ADC ของ ESP32 — calibrate ด้วยค่า dry/wet สอง point
  • รวมกับ DHT22 + BH1750 light sensor → ควบคุม smart irrigation
  • งานวิจัยชายฝั่ง: ปลูกพืชในพื้นที่น้ำเค็มด้วยระบบ IoT ควบคุม
  • ต้นทุนต่ำ <50 บาท/ตัว เหมาะ deploy หลายจุดในไร่นา
ESP32 CPU Temp Sensor Internal / Monitoring

เซ็นเซอร์อุณหภูมิภายใน ESP32 ชิป ใช้ monitor thermal health ของ device โดยไม่ต้องต่อ sensor ภายนอก

  • Library ESPCPUTemp รองรับ ESP32, S2, S3, C3, C6, H2 ทุกรุ่น
  • Auto-detect chip variant พร้อม robust error handling
  • ใช้ trigger thermal throttle หรือ alert เมื่อ device ร้อนเกินไป
  • ดาวน์โหลดได้จาก PlatformIO Registry
  • มีประโยชน์ใน enclosure ปิดหรือ outdoor deployment
