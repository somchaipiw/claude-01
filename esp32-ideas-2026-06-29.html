Meshtastic 2026 — LoRa Mesh Network ที่ไม่ต้องการ Infrastructure
Meshtastic คือเฟิร์มแวร์ open-source สำหรับสื่อสารแบบ off-grid mesh ผ่าน LoRa ไม่ต้องใช้อินเทอร์เน็ต ไม่ต้องมีสัญญาณมือถือ ในปี 2026 รองรับฮาร์ดแวร์กว่า 100 ชนิด และ Web Flasher ใหม่ช่วยให้ Flash firmware ได้ภายใน 5 นาที โดยไม่ต้องเขียนโค้ด รองรับ text messaging, location sharing, telemetry ผ่าน mesh network ที่ self-healing

LoRaWAN · 2026
XIAO ESP32S3 + Wio-SX1262 — Single Channel LoRaWAN Gateway

Seeed Studio ออก guide ใหม่ (พ.ค. 2026) สำหรับการตั้งค่า ESP32-S3 + SX1262 LoRa เป็น Single Channel LoRaWAN Gateway ด้วย SenseCraft App ลงทะเบียนกับ The Things Network หรือ ChirpStack ได้ทันที ค่า path loss ของ LoRa ทำให้สัญญาณส่งได้ไกล 2–15 km ในพื้นที่เปิดโล่ง

  • ราคาชุด XIAO ESP32S3 + SX1262 ≈ $10
  • รองรับ ABP และ OTAA activation
  • Library: EzLoRaWAN (Francois Riotte)
  • รองรับการ deploy sensor node ทั่ว campus หรือฟาร์ม
Touch UI · 2026
LVGL v9 + ESP32 CYD — Professional Touch Screen UI โดยไม่ต้องเขียน CSS

ESP32 Cheap Yellow Display (ESP32-2432S028R) กับ LVGL v9 ทำให้สร้าง touch screen UI ระดับมืออาชีพ ได้บนชิปตัวเดียว ไม่ต้องมี OS รองรับ button, slider, gauge, chart, multi-page navigation ออกแบบ UI ด้วย SquareLine Studio แล้ว export C code เพื่อ compile ใน Arduino IDE

  • Display 2.8" ILI9341 (320×240) built-in touch resistive
  • LVGL handles partial redraw → smooth 30+ fps
  • รองรับ display ขนาด 7" (800×480) บน ESP32-S3
  • ผสาน MQTT ส่งสถานะ relay / sensor ไป Home Assistant
Trail Mate · 2026
Trail Mate Firmware — ESP32 Off-grid Handheld ไม่พึ่งสมาร์ทโฟน

Trail Mate คือ open-source firmware สำหรับ ESP32 + LoRa handhelds โดยเฉพาะ รองรับทั้ง Meshtastic และ MeshCore protocols บนอุปกรณ์เดียวกัน ไม่ต้องพึ่ง Bluetooth หรือสมาร์ทโฟน มี GPS map north-up, direct LoRa text messaging และ GNSS waypoint logging

  • ออกแบบเน้น stability + efficiency สำหรับกิจกรรม outdoor
  • รองรับ interoperability กับ Meshtastic nodes อื่นๆ
  • เหมาะกับ SAR (Search and Rescue) และ wilderness exploration
  • ทำ DIY ได้ด้วย ESP32 + SX1262 + GPS module + OLED
Audio · 2026
ESP32 I2S Audio Stack — Sound Level Meter + FFT อัปเดตล่าสุด Jun 2026

โปรเจกต์ esp32-i2s-slm (Sound Level Meter) อัปเดตล่าสุด 25 มิถุนายน 2026 รองรับ I2S MEMS microphone + digital IIR filters สำหรับ A-weighting และ equalization ให้ค่า dBSPL ที่แม่นยำ ปัจจุบัน DFRobot ออก Fermion I2S MEMS Microphone (SEN0526) พร้อม breakout board ที่ต่อ ESP32 ได้ง่ายมาก

  • Fermion SEN0526: SNR 65 dBSPL, AOP 120 dBSPL
  • Library: ESP32-AudioInI2S รองรับ FFT 64 buckets
  • Visualize audio บน WS2812B LED strip หรือ TFT display
  • ใช้กับ MAX98357A I2S amplifier สำหรับ speaker output

Hackaday · Feb 2026
3D Room Mapping ด้วย VL53L5CX + IMU — โปรเจกต์ซีเรียสที่ใช้ ESP32

นักพัฒนาสร้างระบบ real-time 3D room scanning โดยใช้ VL53L5CX (8×8 ToF matrix) บวก IMU เพื่อเพิ่มข้อมูล orientation ผลลัพธ์ส่งเป็น JSON stream จาก ESP32 ไปยังคอมพิวเตอร์ สำหรับ render point cloud เหมาะสำหรับ robot navigation และ indoor mapping

  • VL53L5CX: 64 zones (8×8), วัดระยะสูงสุด 4 เมตร, 60 Hz
  • ผสาน IMU (ICM-42688) เพื่อ fuse pose + depth data
  • Stream JSON ผ่าน WiFi → Python + Open3D visualization
  • ต้นทุน: VL53L5CX SparkFun Qwiic ≈ $20

GPS · GNSS
ESP32 + NEO-M9N GNSS — GPS Tracker ความแม่นยำสูง Multi-constellation

NEO-M9N รับสัญญาณได้ 4 ระบบพร้อมกัน (GPS, GLONASS, Galileo, BeiDou) ทำให้ lock สัญญาณเร็วกว่า NEO-6M/M8N มาก ต่อ ESP32 ผ่าน UART2 (GPIO 16/17) baudrate 38400 ใช้ Library TinyGPSPlus ส่งพิกัดขึ้น MQTT → แสดงบน Google Maps / Dashboard

  • Accuracy: CEP 2.5 m, Cold start lock < 30 วินาที
  • ใช้ GeoLinker API (Circuit Digest) visualize real-time
  • รองรับ deep sleep → Battery life สูงสุดหลายสัปดาห์
  • ใช้กับ SIM7600 4G ส่งข้อมูลแม้ไม่มี WiFi
ESP32 NEO-M9N TinyGPSPlus MQTT
Industrial IoT
ESP32 Modbus RTU/TCP Gateway — เชื่อม PLC เก่ากับ Cloud ยุคใหม่

ESP32 + LAN8720A Ethernet PHY ทำเป็น Modbus RTU-to-TCP/IP Gateway เชื่อม RS-485 Industrial device (PLC, VFD, sensor) เก่าเข้ากับ WiFi/Ethernet และส่งข้อมูลขึ้น AWS IoT, ThingsBoard, หรือ MQTT broker ได้ทันที เหมาะสำหรับ retrofit โรงงานเก่า

  • รองรับทั้ง Modbus RTU Master/Slave และ TCP Client/Server
  • Library: esp32ModbusMaster, ModbusMaster (Doc Walker)
  • อ่านค่า Float (32-bit) และ Integer จาก holding registers
  • PCB Gerber + schematics พร้อมบน Hackster.io

Spectral · Color
AS7341 Spectral Analyzer Web UI — วัดสีด้วย 11 ช่องสัญญาณผ่าน Browser

AS7341 เป็น 11-channel visible light spectrometer จาก ams OSRAM วัดแสง 415–680 nm ในย่าน visible + NIR + no-filter channel โปรเจกต์บน GitHub (Roger-random) ใช้ ESP32 อ่านค่าผ่าน I2C แล้ว serve web UI ผ่าน WiFi ให้ interact กับ sensor ได้ real-time จาก browser

  • ใช้ตรวจ plant health, display calibration, material QC
  • รองรับ flicker detection (50/60 Hz mains)
  • ESPHome มี component as7341 แล้วใน 2026
  • ราคา breakout ≈ $15 (Adafruit / Waveshare)
