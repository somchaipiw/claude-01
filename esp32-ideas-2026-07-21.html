XIAO ESP32-C5 — Dual-Band Wi-Fi 6 ตัวเล็กที่สุดในโลก

Seeed Studio เปิดตัว XIAO ESP32-C5 บอร์ด IoT ขนาด 21×17.8 mm รองรับ Wi-Fi 6 ทั้ง 2.4 GHz และ 5 GHz, Bluetooth 5.0 LE, Zigbee 3.0 และ Thread — ทุกอย่างในบอร์ดนิ้วหัวแม่มือเดียว!

  • Processor: RISC-V 32-bit @ 240 MHz
  • Memory: 384 KB SRAM + 8 MB PSRAM + 8 MB Flash
  • Wi-Fi: 6 (802.11ax) dual-band 2.4/5 GHz
  • Deep Sleep: เพียง 15 µA (ยอดเยี่ยมสำหรับ battery node)
  • รองรับ: Arduino, PlatformIO, MicroPython, ESP-IDF
  • Matter/Thread: ready for smart home standard
  • ราคา: เพียง $6.90 USD บน Seeed Studio
💡 เหมาะสำหรับ smart home hub ขนาดเล็ก, Matter sensor node, edge gateway ไร้สาย 5GHz

🏭 Industrial

Waveshare ESP32-S3-RS485-CAN — Industrial IoT Gateway

บอร์ด ESP32-S3-RS485-CAN ของ Waveshare คือ industrial control board ที่ออกแบบมาสำหรับโรงงานโดยเฉพาะ มาพร้อม isolated RS485 และ isolated CAN bus ในบอร์ดเดียวกัน รองรับแรงดันไฟฟ้า 7–36V DC

  • Processor: ESP32-S3 dual-core LX7 @ 240 MHz
  • RS485: isolated, UART up to 5 Mbps — รองรับ Modbus RTU
  • CAN/TWAI: isolated, up to 1 Mbps — สำหรับ CAN bus devices
  • Power: 7–36V wide-voltage screw terminal input
  • Wireless: Wi-Fi + Bluetooth 5 LE dual-mode
  • Protection: power isolation + optocoupler isolation
  • ราคา: ~$17.50 USD
⚙️ ใช้เป็น Modbus-to-MQTT gateway เชื่อม PLC/sensor โรงงานเก่าเข้ากับ cloud ได้ทันที
🏠 Smart Home

ESP32-C5 บน ESPHome — Smart Sensor ไร้สาย 5 GHz

ชุมชน ESPHome เริ่มสนับสนุน ESP32-C5 อย่างเป็นทางการ ทำให้ผู้ใช้ Home Assistant สามารถสร้าง sensor node บน Wi-Fi 5 GHz ได้ครั้งแรก — ลดการแออัดของคลื่น 2.4 GHz ที่บ้านที่มีอุปกรณ์มาก

  • ข้อดี 5 GHz: ความหน่วงต่ำกว่า, แออัดน้อยกว่า 2.4 GHz
  • Zigbee/Thread: รองรับ IEEE 802.15.4 ในชิปเดียวกัน
  • Matter: native support, เชื่อมกับ Apple HomeKit, Google Home
  • ESPHome config: ระบุ wifi band: 5ghz เพื่อบังคับใช้ 5 GHz
  • เหมาะกับ: บ้านที่มีอุปกรณ์ IoT มากกว่า 30 ชิ้น
📶 Tip: ใช้ XIAO ESP32-C5 ($6.90) + ESPHome เพื่อสร้าง multi-protocol sensor node ราคาถูกที่สุด

💧 Water IoT

สถานีตรวจคุณภาพน้ำ ESP32 แบบ Real-Time

สร้างระบบตรวจวัดคุณภาพน้ำอัตโนมัติด้วย ESP32 และ sensor หลายชนิด ส่งข้อมูลขึ้น cloud ดูได้จากทั่วโลก เหมาะกับบ่อเลี้ยงปลา, แหล่งน้ำชุมชน หรืองานวิจัย

1
Hardware: ESP32 + pH sensor SEN0161 + TDS sensor SEN0244 + Turbidity SEN0189 + DS18B20 temperature
2
Communication: ส่งข้อมูล MQTT ทุก 5 นาที → Node-RED → InfluxDB
3
Dashboard: Grafana แสดงกราฟ real-time, แจ้งเตือน LINE เมื่อค่าผิดปกติ
4
Power: โซลาร์เซลล์ 5W + แบตเตอรี่ 18650 × 2 ทำงานออฟไลน์ได้
🐟 ประยุกต์: ตรวจ DO (Dissolved Oxygen) สำหรับบ่อเลี้ยงกุ้ง/ปลา แจ้งเตือนทันทีเมื่อออกซิเจนต่ำ

Solar-Powered Field Sensor Node ด้วย ESP32-C5

สร้าง sensor node ทำงานได้เอง 100% ด้วยแสงอาทิตย์ ไม่ต้องเดินสายไฟหรือเปลี่ยนถ่าน เหมาะกับแปลงเกษตร, กล้อง trail cam, สถานีอุตุฯ ห่างไกล

1
Hardware: XIAO ESP32-C5 + MPPT charger (CN3791/BQ25570) + แผงโซลาร์ 5W + 18650 × 1
2
Sensor: BME280 (temp/humidity/pressure) + SHT41 + ไฟฟ้าสถิต AHT21
3
Deep Sleep: ตั้งค่า deep sleep 5 นาที, wake → อ่านค่า → ส่ง MQTT → sleep ใช้ไฟเพียง 15 µA ขณะ sleep
4
Wi-Fi: ใช้ band 5 GHz เพื่อส่งข้อมูลเร็วกว่า, ลดเวลา active → ประหยัดพลังงานมากขึ้น
⚡ ทำงานอัตโนมัติหลายปีโดยไม่ต้องบำรุงรักษา ติดตั้งได้ทุกที่ที่มีแสงแดด
🌱 Smart Farm

ระบบตรวจดินอัจฉริยะ NPK + pH ด้วย ESP32 + Modbus

วัดธาตุอาหารในดิน N (ไนโตรเจน), P (ฟอสฟอรัส), K (โพแทสเซียม) พร้อม pH, EC และอุณหภูมิดิน แนะนำปุ๋ยอัตโนมัติด้วย rule engine ใน Node-RED

1
Sensor: RS485 NPK 7-in-1 Soil Sensor (วัด N, P, K, pH, EC, Temp, Moisture) ราคา ~800-1,500 บาท
2
Protocol: Modbus RTU ผ่าน MAX485 chip → ESP32 UART2 อ่านค่าทุก 30 นาที
3
AI Logic: ตรวจสอบค่าเทียบกับ crop database → แนะนำ "เพิ่มปุ๋ย N 20 kg/ไร่" อัตโนมัติ
4
Display: แสดงผล e-Paper 2.9" หรือ TFT ILI9341 ที่หัวแปลง + Line Notify
🌾 ลดต้นทุนปุ๋ย 20-40% ด้วยการใส่ปุ๋ยตามค่าจริงของดิน ไม่ใส่แบบประมาณ
🏭 Industry 4.0

Modbus-to-MQTT Gateway สำหรับโรงงาน ด้วย ESP32-S3

แปลงอุปกรณ์ Modbus RTU เก่าในโรงงาน (มิเตอร์, PLC, sensor) ให้ส่งข้อมูลขึ้น MQTT broker แบบ real-time ด้วยต้นทุนต่ำและ config ง่าย

1
Hardware: Waveshare ESP32-S3-RS485-CAN (~$17.50) ต่อสาย RS485 Bus กับอุปกรณ์ในโรงงาน
2
Firmware: ใช้ esp-modbus library poll Modbus slave ทุก register ที่ต้องการ ทุก 1 วินาที
3
MQTT: publish ข้อมูลไปยัง broker (Mosquitto/HiveMQ) → subscribe จาก Node-RED หรือ Home Assistant
4
Alert: ตั้ง threshold แจ้งเตือน Telegram/LINE เมื่อ power factor ต่ำ, กระแสเกิน, หรือมอเตอร์หยุด
🔧 Tip: ใส่ termination resistor 120Ω ที่ปลายทั้งสองของ RS485 bus เสมอ เพื่อป้องกัน signal reflection
