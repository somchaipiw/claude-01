Edge AI

ESP32-P4-EYE — Vision AI บนฝ่ามือ

ESP32-P4-EYE คือบอร์ดพัฒนาที่รวม dual-core RISC-V 400 MHz, กล้อง, จอแสดงผล, ไมโครโฟน, และ MicroSD ไว้ในตัวเดียว รองรับ MIPI-CSI / DSI และมี AI Accelerator ในตัวสำหรับรัน YOLOv11-nano ตรวจจับใบหน้า/คน/วัตถุแบบ Real-time ได้ที่ 1080p@30fps

  • รัน TFLite / ESP-DL โดยไม่พึ่ง Cloud
  • รองรับ H.264 Video Encoding ในตัว
  • 32 MB PSRAM, Wi-Fi 6 (ESP32-P4R32)
  • Factory Demo: Photo, Video, Face Detection, YOLOv11
Matter 1.5

Matter 1.5 + ESP32-C6 — Smart Sensor สำหรับบ้าน

Matter 1.5 เพิ่ม device type สำคัญสำหรับ ESP32 developer ได้แก่ กล้อง, Soil Sensor, Energy Management และ TCP Transport ทำให้ส่งข้อมูลภาพและ firmware ผ่าน Matter ได้โดยตรง ESP32-C6 คือตัวเลือกที่ดีที่สุดในการทำ Matter Device ตอนนี้

  • ESP32-C6: Wi-Fi 6 + BLE 5 + Thread/Zigbee ในชิปเดียว
  • TCP Transport: รองรับ payload ขนาดใหญ่ (ภาพ/OTA)
  • รองรับ Apple Home, Google Home, Amazon Alexa
  • เซ็นเซอร์แนะนำ: BME280 + SCD41 CO₂
ESP32-H2

ESP32-H2 + ESPHome Thread — Sensor ประหยัดพลังงาน

ESP32-H2 เป็น Espressif SoC ตัวแรกที่มี IEEE 802.15.4 บนฮาร์ดแวร์ รองรับทั้ง Zigbee 3.0 และ Thread 1.3 พร้อมกัน ทำให้เหมาะมากสำหรับ sensor node ที่ทำงานบนแบตเตอรี่ร่วมกับ ESPHome Thread Border Router

  • Ultra-low power: ลึกกว่า ESP32-C3 ในโหมด sleep
  • M5Stack NanoH2: dev kit ขนาดเล็กมาก
  • ใช้คู่กับ ESP32-C6 เป็น Border Router ได้ทันที
  • ESPHome รองรับ Thread Border Router native แล้ว

TinyML

ระบบ Home Security ออฟไลน์ด้วย ESP32-S3 + TinyML

สร้างกล้องรักษาความปลอดภัยที่รัน ML Model ตรวจจับบุคคลและความเคลื่อนไหว บน ESP32-S3 โดยตรง ไม่ต้องพึ่ง Cloud เลย ใช้ Vector Instructions ของ S3 เร่งความเร็ว inference ให้เร็วขึ้น 3-5 เท่าเทียบกับ ESP32 ธรรมดา

  • Model: MobileNetV2 INT8 quantized → <200KB
  • แจ้งเตือนผ่าน MQTT/HTTP เมื่อพบคน
  • บันทึก snapshot ลง SD card อัตโนมัติ
  • ความล่าช้า inference: ~50ms บน S3

Mesh Network

Self-Healing IoT Mesh ด้วย ESP-NOW (ไม่ต้องมี Wi-Fi Router)

สร้างเครือข่ายเซ็นเซอร์แบบ Mesh ที่ทนทานต่อ node เสียโดยใช้ ESP-NOW Protocol ซึ่ง Espressif พัฒนาขึ้นมาโดยเฉพาะ Node แต่ละตัวรู้จัก neighbor ของตัวเองและส่งข้อมูลผ่าน multi-hop ได้ เหมาะสำหรับเกษตรอัจฉริยะ, โรงงาน, หรือบ้านขนาดใหญ่

  • Library แนะนำ: PainlessMesh, ZHNetwork
  • ช่วง: สูงสุด 200m ต่อ hop (outdoor)
  • Payload สูงสุด 250 bytes ต่อ packet
  • ดูข้อมูลแบบ Real-time บน Node-RED Dashboard
Air Quality

Project Aura v1.1 — เครื่องวัดคุณภาพอากาศ Open Source

Project Aura คือ open-source air quality monitor ที่ใช้ ESP32-S3 พร้อม Sensirion sensor คุณภาพสูง หน้าจอ IPS 4.3" และ ต่อเชื่อม Home Assistant ผ่าน MQTT ได้ทันที เวอร์ชัน 1.1 เพิ่ม CO Sensor (DFRobot SEN0466) ใหม่

  • Sensors: PM2.5, CO₂, TVOC, Temp/RH, CO
  • Display: 4.3" IPS Touchscreen (480×272)
  • Assembly แบบไม่ต้องบัดกรี (solderless)
  • Source: hackster.io/21cncstudio
Predictive

Predictive Maintenance ด้วย TinyML — ทำนายวาล์วเสีย

ใช้ ESP32 + accelerometer/pressure sensor เก็บ vibration signature ของวาล์ว แล้วรัน anomaly detection model บนตัวเพื่อทำนายล่วงหน้าก่อนวาล์วพัง ไม่ต้องส่งข้อมูลไป Cloud เลย ลดต้นทุน bandwidth และ latency

  • Algorithm: Edge Impulse Anomaly Detection
  • Sensor: MPU6050 + ADS1115 pressure
  • Accuracy: ~92% ในการทดสอบจริง
  • แจ้งเตือน Telegram/LINE เมื่อพบ anomaly
Smart Home

Matter Soil Sensor — สำหรับต้นไม้อัจฉริยะ

Matter 1.5 เพิ่ม Soil Sensor device type ทำให้สามารถสร้างเซ็นเซอร์ดิน ที่ทำงานร่วมกับ Apple Home, Google Home และ Amazon Alexa ได้โดยตรง ใช้ ESP32-C6 + Capacitive Soil Moisture Sensor + BME280 เป็นเซ็ตพื้นฐาน

  • วัด: ความชื้นดิน, อุณหภูมิ, ความชื้นอากาศ, ความดัน
  • ทำงานบนแบตเตอรี่ AA 2 ก้อนนาน 6-12 เดือน
  • โปรโตคอล Thread ผ่าน ESP32-H2 ประหยัดพลังงานกว่า
  • esp-matter SDK พร้อมใช้งานบน GitHub Espressif
Electronic Nose

AI Electronic Nose — ดมกลิ่นด้วย TinyML

สร้าง "จมูกอิเล็กทรอนิกส์" โดยใช้ gas sensor array หลายตัวร่วมกับ ML classifier รันบน ESP32-S3 เพื่อแยกแยะกลิ่นหรือก๊าซต่างๆ ประยุกต์ใช้ได้ใน food quality control, ตรวจสอบอากาศ หรือ fire detection

  • Sensors: MQ-series array (MQ-2, MQ-7, MQ-135, MQ-8)
  • Feature extraction: FFT บน sensor signal
  • Classifier: Random Forest / SVM ขนาดเล็ก
  • Accuracy: 85-95% ขึ้นกับจำนวน class ที่ฝึก

Sensors & Modules น่าสนใจ

Component ใหม่ที่ ESPHome และ Arduino รองรับในช่วงนี้

ESPHome ใหม่

ScioSense UFM-01 — Ultrasonic Flow Meter

Flow meter แบบ ultrasonic รุ่นใหม่ที่ ESPHome 2026.7 รองรับ วัดได้ทั้ง อัตราการไหล (L/min), ปริมาณสะสม, และอุณหภูมิน้ำ พร้อม diagnostic binary sensors ผ่าน UART ไม่ต้องตัดท่อ (clamp-on design)

  • Interface: UART (115200 baud)
  • วัดอุณหภูมิน้ำ ±0.5°C ในตัว
  • Binary sensor: water presence, flow direction
  • เหมาะสำหรับ: ระบบน้ำ solar, สระว่ายน้ำ, ชลประทาน
ESPHome ใหม่

QMI8658 — 6-Axis IMU ความแม่นยำสูง

IMU 6 แกน (Accelerometer + Gyroscope) พร้อม on-chip temperature sensor ที่ ESPHome 2026.7 รองรับ native ผ่าน I2C บน Waveshare ESP32-S3 boards และบอร์ดอื่นๆ ที่ใช้ชิปนี้

  • 6 DOF: Accel ±2/4/8/16g, Gyro ±16/32/64/128°/s
  • Output Data Rate: สูงสุด 7174 Hz
  • Built-in Motion Platform สำหรับ pedometer
  • ใช้ทำ: drone stabilization, gesture detection, wearable
Display

SH8601 AMOLED 1.64" — สำหรับ ESP32-S3

จอ AMOLED ขนาด 1.64 นิ้วบน Waveshare ESP32-S3-Touch-AMOLED ที่ ESPHome รองรับแล้ว ให้ภาพคมชัดสีสดในขนาดเล็ก เหมาะสำหรับ wearable, smart badge, desktop widget

  • ความละเอียด: 412×412 pixels, true black AMOLED
  • Interface: SPI (QSPI-compatible)
  • Touch: Capacitive multi-touch
  • กินไฟต่ำกว่า IPS/LCD แบบเดิมมาก
Sensor Module

DFRobot SEN0466 — CO Sensor สำหรับ Air Quality

เซ็นเซอร์วัดก๊าซ Carbon Monoxide (CO) ที่ Project Aura v1.1 เลือกใช้ ออกแบบมาสำหรับ indoor air quality monitoring โดยเฉพาะ เหมาะต่อสุขภาพเพราะ CO จากเตาแก๊สหรือรถยนต์เป็นอันตรายถึงชีวิต

  • วัด CO: 0-1000 ppm, resolution 1 ppm
  • Interface: I2C หรือ Analog output
  • แจ้งเตือนทันทีเมื่อ CO เกินระดับอันตราย
  • รองรับ ESPHome + Home Assistant โดยตรง
Display

Pixoo 64 — LED Matrix สำหรับ ESP32

LED Matrix 64×64 pixel RGB ที่ ESPHome 2026.7 รองรับ native เชื่อมต่อผ่าน SPI ควบคุม brightness ผ่าน light platform และแสดงผล pixel art, animation, หรือ sensor data แบบ colorful ได้

  • 64×64 = 4,096 RGB pixels ต่อ panel
  • Interface: SPI + เชื่อม chain หลาย panel ได้
  • ใช้ทำ: status display, ambient visualizer, game
  • Image API ผ่าน ESPHome draw/image component
Connectivity

M5Stack NanoH2 — ESP32-H2 ขนาดเล็กสุด

Dev kit ESP32-H2 ขนาดเล็กมากจาก M5Stack รองรับ Zigbee 3.0, Thread 1.3 และ Matter ในตัวเดียว ออกแบบมาสำหรับ battery-powered IoT sensor node ที่ต้องการ wireless protocol ทั้งสองแบบ

  • ขนาด: ~24×24mm (ultra-compact)
  • รองรับ Zigbee Coordinator/Router/End Device
  • Thread End Device พร้อม low-power deep sleep
  • แหล่งซื้อ: shop.m5stack.com
