🔥 HOT

ESP32-P4 Revision 3.x — Stable 400 MHz + Power Rail ใหม่

Espressif ออก ESP32-P4 revision 3.0 อย่างเป็นทางการในปี 2026 พร้อมการเปลี่ยนแปลงสำคัญ: clock speed คงที่ที่ 400 MHz (เสถียรกว่าเดิมมาก), power rail ใหม่ที่ต้องออกแบบ PCB ใหม่, และ multimedia module (ISP + PPA) ได้รับการอัปเกรดใหญ่

ไฮไลต์สำคัญ: H.264 encoding สูงสุด 1080p@30fps, 32 MB embedded PSRAM, Hardware CNN accelerator สำหรับ TFLite Micro และ ONNX Micro Runtime, รองรับ MIPI-CSI กล้องและ MIPI-DSI จอแสดงผล

✨ NEW

ESP32-C6 Zigbee Native Mode — Smart Home ไม่ต้องพึ่ง Hub เพิ่ม

ESP32-C6 รองรับ Zigbee 3.0, Thread 1.3, Wi-Fi 6 และ BLE 5 บนชิปเดียว แต่ข้อสำคัญที่ต้องรู้: ใช้งานได้ทีละ mode เลือก flash firmware ให้ตรงกับ protocol ที่ต้องการ

ในปี 2026 ทั้ง Waveshare และ Seeed Studio ออกบอร์ด ESP32-C6 สำหรับ DIY Thread Border Router ราคาไม่ถึง 200 บาท เหมาะสำหรับ Home Assistant ที่ต้องการขยาย Thread network

💡 TECHNIQUE

ESP32 ULP Coprocessor — แบตเตอรี่อยู่ได้ 12 ปีด้วย Deep Sleep

ULP (Ultra Low Power) คือ processor เล็กๆ ขนาด 8 MHz ภายใน ESP32 ที่ทำงานได้ แม้ core หลักปิดอยู่ ใช้ไฟเพียง 150 µA เท่านั้น

ตัวอย่างจริง: sensor node ใช้ถ่าน 18650 สองก้อน (6,000 mAh) ULP อ่าน ADC ทุก 10 วินาที ถ้าค่าเกิน threshold ถึงจะ wake CPU ขึ้นส่ง WiFi — ใช้ไฟเฉลี่ย 1.11 mAh/วัน แบตเตอรี่อยู่ได้ประมาณ 4,320 วัน (~12 ปี)

🛠️ DIY

Wearable Health Monitor ด้วย ESP32 + MAX30102 + AD8232

สร้างสายรัดข้อมือ / patch monitor ราคาประหยัดที่วัด Heart Rate, SpO2 (ออกซิเจนในเลือด) และ ECG 1-lead พร้อมส่งข้อมูลขึ้น cloud แบบ real-time ผ่าน BLE ไปยังมือถือ

อุปกรณ์ที่ใช้: ESP32-S3 Mini + MAX30102 (PPG sensor, I²C) + AD8232 (ECG analog front-end) + OLED 0.96" + LiPo 180mAh ประยุกต์ได้กับ ESP-IDF และ Arduino framework

// MAX30102 อ่าน Heart Rate + SpO2 #include "MAX30105.h" #include "heartRate.h" #include "spo2_algorithm.h" MAX30105 particleSensor; int32_t spo2, heartRate; int8_t validSPO2, validHeartRate; particleSensor.begin(Wire, I2C_SPEED_FAST); particleSensor.setup(60, 4, 2, 100, 411, 4096); maxim_heart_rate_and_oxygen_saturation( irBuffer, bufferLength, redBuffer, &spo2, &validSPO2, &heartRate, &validHeartRate );

🛠️ DIY

Air Quality Station แบบ Pro ด้วย 3 Sensors + Dashboard

รวม sensor สามตัวเข้าด้วยกัน: BME690 (อุณหภูมิ/ความชื้น/ความดัน/VOC gas), BMV080 (PM2.5 laser particle), SCD41 (CO₂ photoacoustic) ส่งข้อมูลทุก 60 วินาทีไปยัง InfluxDB + Grafana dashboard ผ่าน ESP32-S3 Mini

ต่อเพิ่ม e-paper display ขนาด 2.9" แสดง AQI index สีเขียว/เหลือง/แดง พร้อม Wi-Fi OTA update — ใช้ไฟ USB-C จากธนาคารพลังงาน

🛠️ DIY

Smart Plant Monitor ด้วย ULP — แบตอยู่ 2+ ปี ไม่ต้องชาร์จ

ใช้ ULP coprocessor อ่าน soil moisture sensor ทุก 10 วินาที หากดินแห้งเกิน threshold ถึงจะ wake CPU ขึ้นเชื่อม Wi-Fi ส่งแจ้งเตือน Line Notify และรดน้ำอัตโนมัติผ่าน relay

CPU "หลับ" ตลอดเวลาที่ดินชื้นพอ ทำให้ประหยัดพลังงานมหาศาล — ใช้ถ่าน AA สองก้อนได้นานกว่า 2 ปี เหมาะกับกระถางต้นไม้กลางแจ้ง หรือเรือนเพาะชำ

ULP Coprocessor Soil Moisture Line Notify Auto Watering 2 ปีต่อชาร์จ
🔥 ADVANCED

ESP32-P4 AI Vision Camera — Face Detection + Object Count ไม่ต้องพึ่ง Cloud

ใช้ ESP32-P4-Function-EV-Board (จอ 7" 1024×600, กล้อง MIPI-CSI 2MP, ESP32-C6 Wi-Fi) รัน TFLite Micro model สำหรับนับคนในห้อง / ตรวจจับหน้า / จำแนกวัตถุ แบบ offline ไม่ต้องส่งภาพออก internet

CNN hardware accelerator + 32 MB PSRAM ของ P4 รองรับ model ขนาด ~500 KB ได้ดี เหมาะสำหรับ Smart Building access control หรือ retail people counter ราคาต่ำ
