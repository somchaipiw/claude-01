NEW CHIP
ESP32-S31 — Wi-Fi 6 + Bluetooth 5.4 + BLE Mesh 1.1
Espressif เปิดตัว ESP32-S31 ชิปตัวใหม่ล่าสุด รองรับ Wi-Fi 6 (802.11ax), Bluetooth 5.4 และ BLE Mesh 1.1 สำหรับระบบ Smart Home และ Industrial IoT ขนาดใหญ่ที่ต้องการ self-healing network หลายร้อย node
  • Dual-Core RISC-V, Wi-Fi 6 (802.11ax) 2.4 GHz
  • Bluetooth 5.4 + BLE Mesh 1.1 รองรับสูงสุด 32,767 nodes
  • Advanced HMI — เหมาะสร้าง touchscreen panel
  • Fast Provisioning: เพิ่ม 100 nodes ได้ใน 50 วินาที
✨ เหมาะกับ Smart Factory, Smart Lighting และ Building Automation
ROBOTICS
ESP32-P4 Intelligent Robotic Arm — Kinematics + Vision บนชิปเดียว
Espressif สาธิต Robotic Arm อัจฉริยะบน ESP32-P4 ที่รัน Forward & Inverse Kinematics พร้อม Computer Vision บนตัวชิปเดียว ไม่ต้องพึ่ง external computing แสดงให้เห็นที่ Embedded World 2026
  • คำนวณ Kinematics ด้วย D-H Parameters บนตัวชิป
  • USB Camera + Color-Detect Model จับวัตถุตามสีได้แม่นยำ
  • Calibration Matrix สำหรับ coordinate transform แขนกล
  • Remote Control ผ่าน Wi-Fi — ควบคุมจากทุกที่
🏭 Industrial prototype ที่รันได้ด้วยชิปราคา $10
LORAWAN
Gateway-Free LoRa Mesh — Self-Healing Network ไร้โครงสร้างพื้นฐาน
งานวิจัยใหม่ปี 2026 สร้างเครือข่าย LoRa Mesh บน ESP32 ที่ไม่ต้องพึ่ง Gateway มี self-healing โดย neighbor-based routing, hop-by-hop acknowledgment และ controlled retransmission เหมาะสำหรับพื้นที่ห่างไกล
  • ไม่ต้องการ Gateway — node ทำหน้าที่ relay ได้ทันที
  • Neighbor-based routing เลือกเส้นทางเองแบบอัตโนมัติ
  • Hop-by-hop ACK ป้องกันข้อมูลสูญหาย
  • รัน Meshtastic mesh routing + Wi-Fi MQTT bridge พร้อมกัน
📖 Published in MDPI Sensors, Vol. 25, 2025
ENERGY
Solar-Piezo Hybrid Energy Harvesting — IoT Node ไร้แบตเตอรี่
เทคโนโลยี Hybrid Energy Harvesting รวม Solar + Piezoelectric ทำให้ ESP32 Node ทำงานได้โดยไม่ต้องชาร์จแบตเตอรี่เลย เหมาะกับ Smart Agriculture และ Industrial Vibration Monitoring ในจุดที่สายไฟเข้าไม่ถึง
  • Perovskite Indoor Solar Cell ประสิทธิภาพ 32–38% ใต้แสง LED
  • TENG (Triboelectric Nanogenerator) จากการสั่นสะเทือน <10 Hz
  • ESP32 Deep Sleep duty-cycle: ตื่น 1 วินาที หลับ 60 วินาที
  • Average Power ~1–2 mW — เพียงพอสำหรับ sensing + transmit
☀️ แหล่งพลังงาน 2 แหล่งครอบคลุมกลางวัน (solar) และกลางคืน (vibration)

Sensors & Modules น่าสนใจสำหรับ ESP32

Semtech SX1262
LoRa Transceiver Module สำหรับ Long-Range Communication
ย่านความถี่: 863–928 MHz
ระยะส่ง: 5–15 กม. (ชนบท/เปิดโล่ง)
ความไว: -148 dBm (SF12, BW125)
Interface: SPI — ใช้ได้กับ ESP32 ทุกรุ่น
รองรับ: LoRaWAN, Meshtastic, Custom Mesh
ADXL345
3-Axis Digital Accelerometer สำหรับ Vibration & Motion
ช่วงวัด: ±2g / ±4g / ±8g / ±16g
Resolution: 13-bit
Output Rate: สูงสุด 3200 Hz
Interface: I2C / SPI
ใช้กับ: Vibration Monitor, Tilt, Free-fall Detection
INA3221
Triple-Channel Current & Voltage Monitor
ช่องวัด: 3 channels พร้อมกัน
ช่วงแรงดัน: 0–26 V ต่อช่อง
ความละเอียด: ±1% ทั่วไป
Interface: I2C (address 0x40–0x43)
ใช้กับ: Smart Power Meter, Battery Monitor, Solar Tracking
RAK11160
Dual-Core LoRaWAN Module (ESP32-C2 + STM32WLE5)
Core: ESP32-C2 (Wi-Fi/BLE) + STM32WLE5 (LoRa)
Protocol: LoRaWAN 1.0.x / 1.1
กินไฟ: Ultra-low power สำหรับ battery node
Interface: UART / SPI ควบคุม LoRa ด้วย AT command
ใช้กับ: Long-range Battery-powered Sensor Node
