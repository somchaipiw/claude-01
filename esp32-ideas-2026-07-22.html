CES 2026 · Wi-Fi 6E

ESP32-E22 — ชิป Wi-Fi 6E ตัวแรกจาก Espressif

Espressif เปิดตัว ESP32-E22 ซึ่งเป็น Co-Processor Wi-Fi 6E ตัวแรกในตระกูล ESP32 รองรับคลื่นความถี่ 3 ย่าน (2.4 / 5 / 6 GHz) ด้วย Channel Bandwidth 160 MHz และ 2×2 MIMO ที่ให้ Throughput สูงสุดทางทฤษฎีถึง 2.1 Gbps เหมาะสำหรับ Wireless Gateway, Smart TV, และ Industrial IoT รุ่นใหม่

  • CPU: Dual-core RISC-V สูงสุด 500 MHz · 1 MB On-chip RAM
  • Wi-Fi: 6E Tri-band (2.4/5/6 GHz) · 160 MHz BW · 2×2 MIMO
  • Throughput: สูงสุด 2.1 Gbps (PHY rate)
  • Use Case: Gateway, Video Streaming, High-density AP
🌐 ชิปตัวแรกในโลก IoT ราคาประหยัดที่รองรับ 6 GHz Band — เปิดประตูสู่เครือข่ายไร้การรบกวน
CES 2026 · BLE Ultra-Low Power

ESP32-H21 — Bluetooth LE MCU กินไฟน้อยที่สุด

เปิดตัวพร้อมกันกับ ESP32-E22 ที่ CES 2026 ESP32-H21 คือ MCU Bluetooth LE 5.x ที่ออกแบบมาเพื่ออุปกรณ์ที่ใช้ถ่านหรือพลังงานต่ำเป็นหลัก เช่น Wearables, Smart Tags, Sensor Beacons และ Asset Tracking มุ่งแข่งขันกับ Nordic nRF52 Series โดยตรงในตลาด BLE-only devices

  • Protocol: Bluetooth LE 5.x (Low Energy) · พร้อม Thread
  • Target: Battery-powered IoT · Wearables · Asset Tags
  • Deep Sleep: บริโภคพลังงานระดับ µA ขณะ Sleep
  • Toolchain: ESP-IDF · Zephyr RTOS รองรับ
🔋 ออกแบบมาเพื่อแข่งกับ Nordic nRF52 — เพิ่มตัวเลือก Ultra-Low Power ให้นักพัฒนาไทย
เมษายน 2026 · Wi-Fi 6 + GbE

ESP32-S31 — RISC-V Dual-Core พร้อม Gigabit Ethernet

ESP32-S31 ที่ Hackaday รายงานในเดือนเมษายน 2026 คือก้าวใหม่ของตระกูล S-Series รวม Wi-Fi 6 เข้ากับ Gigabit Ethernet Controller ในชิปเดียว ลดต้นทุนการออกแบบ PCB สำหรับ Edge Computing และ Industrial Gateway ลงได้มาก เหมาะสำหรับ NAS, Industrial Controller, และ 4G/5G Router ราคาประหยัด

  • CPU: Dual-core RISC-V (ความเร็วสูง)
  • Wi-Fi: Wi-Fi 6 (802.11ax) บนย่าน 2.4/5 GHz
  • Ethernet: Gigabit Ethernet Controller on-chip
  • Use Case: Edge Gateway · Industrial IoT · Smart NAS
🏭 ชิปตัวแรกจาก Espressif ที่มี GbE on-chip — ลด BOM ของ Industrial Gateway ได้มากกว่า 30%
Microbots · ขนาดจิ๋ว

CodeCell C6 Drive — ESP32-C6 หุ่นยนต์ครบในบอร์ดเดียว

CodeCell C6 Drive จาก Microbots รวม Dual H-Bridge Motor Driver, 9-Axis IMU (BNO085), Proximity Sensor (VCNL4040) และ LiPo Battery Charging ไว้ในบอร์ดขนาดเพียง 22.5 × 18.5 × 9.4 มม. น้ำหนัก 3.4 กรัม บน ESP32-C6 ที่รองรับ Wi-Fi 6, BLE 5, Zigbee และ Thread

  • ขนาด: 22.5 × 18.5 × 9.4 มม. · น้ำหนัก 3.4 กรัม
  • Motor: Dual H-Bridge ในตัว — ขับ DC Motor / Buzzer / Actuator
  • IMU: BNO085 9-Axis Absolute Orientation (Sensor Fusion)
  • Sensor: VCNL4040 Proximity + Ambient Light
  • Power: LiPo Charging on-board · รองรับ Arduino & MicroPython
🤖 บอร์ดหุ่นยนต์ที่เล็กที่สุดที่ยังมี IMU + Motor Driver + BLE + Wi-Fi 6 ครบในตัว

ESP32-S3 · TinyML · Voice AI

Voice Assistant ออฟไลน์ด้วย ESP32-S3 + INMP441 + Edge Impulse

สร้างผู้ช่วยเสียงที่ทำงาน 100% ออฟไลน์ — ตรวจจับ Wake Word และจดจำคำสั่งเสียงในภาษาไทย โดยไม่ส่งข้อมูลเสียงไปยังคลาวด์ เหมาะสำหรับบ้าน Smart Home ที่เน้นความเป็นส่วนตัว

  • ต่อ INMP441 I2S MEMS Mic กับ ESP32-S3 (GPIO สำหรับ BCK, WS, DATA)
  • ฝึก TinyML Model ด้วย Edge Impulse Studio บนชุดข้อมูลเสียงภาษาไทย
  • Export เป็น Arduino Library แล้ว Deploy ลง ESP32-S3 บน PSRAM 8 MB
  • Trigger Action (เปิดไฟ / เล่นเพลง / ส่งข้อมูล MQTT) เมื่อจดจำคำสั่งได้

Heltec V4 · LoRa · Meshtastic

เครือข่าย Meshtastic ระยะไกลสำหรับพื้นที่ห่างไกล

สร้างเครือข่ายสื่อสาร Mesh แบบ Off-grid ด้วย ESP32-S3 + SX1262 LoRa บน Heltec WiFi LoRa 32 V4 ไม่ต้องพึ่ง Wi-Fi หรือ 4G ครอบคลุมระยะหลายกิโลเมตร เหมาะสำหรับพื้นที่ชนบท ป่าเขา หรือสถานการณ์ฉุกเฉิน

  • ติดตั้ง Meshtastic Firmware บน Heltec WiFi LoRa 32 V4 (ESP32-S3 + SX1262)
  • ตั้งค่า LoRa Region Thailand (AS923) ผ่าน Meshtastic App บนมือถือ
  • ติดตั้ง Nodes หลายจุด เพื่อสร้าง Mesh ขยายสัญญาณอัตโนมัติ
  • ผสาน MQTT Bridge เพื่อส่งข้อมูล Sensor ขึ้น Dashboard ส่วนกลาง
SX1262 AS923 Meshtastic Off-Grid Emergency Comm
ESP32-P4 · Robotics · ESP-NOW

แขนกลอัจฉริยะ Leader-Follower ด้วย ESP32-P4 + ESP-NOW

จำลองระบบ Teleoperation แบบมืออาชีพ — ใช้แขนกลขนาดเล็ก (Leader) บน ESP32-P4 เป็นตัวควบคุม ส่งข้อมูล Joint Position ผ่าน ESP-NOW ไปยังแขนกลอีกตัว (Follower) แบบ Real-time ทดสอบแนวคิด Imitation Learning สำหรับ Robot

  • ติดตั้ง AS5600 Magnetic Encoder บนแต่ละ Joint ของแขนกล Leader
  • ESP32-P4 อ่านมุม Joint ทุก 10 ms แล้วส่งผ่าน ESP-NOW (Latency < 2 ms)
  • Follower ESP32-P4 รับค่าแล้วขยับ Servo ตาม PID Controller
  • บันทึก Trajectory ลง Flash เพื่อ Playback อัตโนมัติ (Teach & Replay)
ESP32-P4 AS5600 ESP-NOW PID Control Teach & Replay
ESP32-C3 · Energy Harvesting · Solar

Sensor Node ไร้สายไม่ต้องเปลี่ยนถ่านด้วย TI BQ25504

สร้าง IoT Sensor Node ที่ดูดพลังงานจากสิ่งแวดล้อม (Solar, RF, Thermal) มาชาร์จ Supercapacitor โดยไม่ต้องมีถ่านหรือแบตเตอรี่ เหมาะกับ Deployment ระยะยาวในที่ที่เข้าถึงได้ยาก เช่น ป่าชายเลน หรือ ฟาร์มปลา

  • ต่อแผง Solar Cell 5 V ขนาดเล็ก เข้า TI BQ25504 Energy Harvesting IC
  • BQ25504 ชาร์จ Supercapacitor 1–10 F แบบ Boost Converter (อินพุต > 100 mV)
  • ESP32-C3 ตื่นจาก Deep Sleep (15 µA) ทุก 5 นาที อ่านค่า Sensor
  • ส่งข้อมูลผ่าน Wi-Fi + MQTT แล้วกลับเข้า Deep Sleep ทันที

60 GHz Radar · ทะลุผ่านผนัง

Acconeer XM125 — เรดาร์ Pulsed Coherent 60 GHz ระยะ 20 เมตร

SparkFun ออก Qwiic Sensor บน XM125 ที่ใช้ Pulsed Coherent Radar (PCR) ความถี่ 60 GHz ตรวจจับวัตถุได้ระยะถึง 20 เมตร และแยกแยะได้ว่าเป็นวัตถุนิ่ง หรือเคลื่อนที่ พร้อมวัดความเร็ว สามารถ "มองทะลุผนัง" บางๆ ได้ ต่อกับ ESP32 ผ่าน I²C หรือ UART

  • เทคโนโลยี: 60 GHz Pulsed Coherent Radar (PCR)
  • ระยะ: สูงสุด 20 เมตร · แยกแยะความเร็วได้
  • ฟีเจอร์: ทะลุผนัง / กระจก / พลาสติก บาง ๆ
  • Interface: I²C / UART · Qwiic-compatible
  • Use Case: ตรวจจับการบุกรุก, Presence Detection, Gesture
9-Axis IMU · Sensor Fusion

BNO085 — 9-Axis Absolute Orientation IMU พร้อม Fusion ในตัว

IMU ระดับมืออาชีพที่มี Sensor Fusion Engine อยู่ในชิป ให้ค่า Quaternion และ Euler Angle ที่เสถียรโดยไม่ต้องเขียน Kalman Filter เอง ใช้ใน CodeCell C6 Drive และ Drone, Robotic Arm, AR Glasses หลายตัว ต่อกับ ESP32 ผ่าน I²C หรือ SPI

  • Axes: Accelerometer 3 + Gyroscope 3 + Magnetometer 3 = 9 Axes
  • Output: Quaternion, Euler Angles, Linear Accel, Gravity Vector
  • Fusion: CEVA SH-2 Sensor Hub on-chip ไม่ต้องคำนวณเอง
  • Interface: I²C / SPI / UART
  • Use Case: Drone, Robotic Arm, Wearable, AR/VR
I2S MEMS Microphone

INMP441 — I2S MEMS Mic คุณภาพสูงสำหรับ Voice AI

ไมโครโฟน Omnidirectional แบบดิจิทัล (I2S) ที่นิยมใช้กับ ESP32 มากที่สุด สัญญาณเสียงสะอาด ไม่มี ADC Noise เชื่อมต่อผ่าน I2S Port โดยตรง เหมาะสำหรับงาน Wake Word Detection, Voice Command และ Audio Logging

  • Type: MEMS Omnidirectional I2S Digital Microphone
  • SNR: 61 dB(A) — เสียงสะอาดมากสำหรับ ML
  • Interface: I2S (ต่อ BCK, WS, SD กับ ESP32 ตรง)
  • Frequency Response: 60 Hz – 15 kHz
  • Use Case: Wake Word, Voice Command, Noise Monitor
24 GHz mmWave · Multi-Human

Ai-Thinker RD-03D — mmWave Radar ตรวจจับมนุษย์ 5 คนพร้อมกัน

Radar ความถี่ 24 GHz สำหรับ Human Presence Detection ที่ตรวจจับได้พร้อมกัน สูงสุด 5 เป้าหมาย พร้อมวัดระยะทาง ความเร็ว และตำแหน่ง X-Y มีโหมด Static Detection (คนนิ่ง) ด้วย เชื่อมต่อกับ ESP32 ผ่าน UART เหมาะสำหรับ Smart Home ที่ต้องการ Privacy มากกว่ากล้องวงจรปิด

  • ความถี่: 24 GHz mmWave
  • Target Tracking: สูงสุด 5 คนพร้อมกัน (ระยะ, ความเร็ว, X-Y)
  • Static Detection: ตรวจจับคนนิ่ง (หายใจ, ชีพจร)
  • Interface: UART · เชื่อมต่อ ESP32 ง่าย
  • Use Case: Smart Lighting, Security, Occupancy Counter
