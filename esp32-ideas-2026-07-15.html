NFC/RFID Security MQTT
ESP32-S3 + PN532 Smart Access Control พร้อม Cloud Log
ระบบ Access Control ด้วย NFC/RFID รูปแบบ Professional — PN532 อ่านการ์ด Mifare/NFC, ESP32-S3 ตรวจสอบสิทธิ์ และ Log ประวัติเข้า-ออกไปยัง Firebase พร้อมแจ้งเตือน MQTT และ LINE Notify แบบ Real-time
1
Reader: PN532 NFC/RFID Module (I2C/SPI) รองรับ ISO14443A, Mifare Classic/Ultralight, NTAG
2
Lock: Relay Module ต่อ Electric Door Strike หรือ Solenoid Lock 12V
3
Feedback: NeoPixel RGB LED + Buzzer บอกสถานะ Granted/Denied
4
Cloud: Log UID + Timestamp ไป Firebase, ดู Dashboard ผ่าน Web App
// ตรวจสอบสิทธิ์ NFC card if (nfc.readPassiveTargetID(PN532_MIFARE_ISO14443A, uid, &uidLength)) { String cardUID = uidToString(uid, uidLength); bool granted = authorizedUIDs.count(cardUID) > 0; setAccessResult(granted); logToFirebase(cardUID, granted); }

AS7341
ams OSRAM — 11-Channel Spectral Color Sensor
Spectrometer ขนาดจิ๋วที่วัดแสงได้ 11 ช่องสเปกตรัม ครอบคลุม 415–680 nm (Visible) + NIR + Clear Channel ผ่าน I2C ใช้งานได้กับ ESP32, Raspberry Pi, Arduino ใช้ในงาน Plant Health Monitoring, Display Calibration, Color QC

รองรับ ESPHome โดยตรง — เพิ่ม config ไม่กี่บรรทัดก็ใช้ได้ใน Home Assistant ทันที
Adafruit Breakout Board ~$17.95 USD

pH + TDS Module
DFRobot / Atlas Scientific — Water Quality Sensors
ชุด Sensor วัดคุณภาพน้ำสำหรับ ESP32 ประกอบด้วย:
• pH Electrode (0–14 pH, ±0.1 accuracy)
• TDS Sensor (0–1000 ppm วัด Total Dissolved Solids)
• Turbidity Sensor (0–3000 NTU)
DFRobot Gravity Series ใช้ Analog Output ต่อกับ ADC ของ ESP32 ได้โดยตรง — ติดตั้งง่าย
DFRobot Gravity pH Kit ~$29.90 USD
📳
PN532
NXP — NFC/RFID Reader/Writer Module
Module NFC/RFID ยอดนิยม รองรับหลาย Protocol:
• ISO 14443A/B (Mifare Classic, Ultralight, NTAG)
• FeliCa, ISO 18092 (NFCIP-1 P2P Mode)
• อ่าน/เขียน Card และ Emulate Card ได้
เชื่อมต่อผ่าน I2C, SPI หรือ HSU (UART) ตามความต้องการ มี Library ทั้ง Arduino และ ESP-IDF
Generic PN532 Module ~$3–$6 USD
📶
SX1262
Semtech — Sub-GHz LoRa / FSK Transceiver
LoRa Transceiver รุ่นใหม่ที่มาแทน SX1276 มีประสิทธิภาพสูงกว่ามาก:
• TX Power สูงสุด +22 dBm
• รองรับ LoRa, LoRaWAN, Long Range FHSS
• ใช้พลังงานน้อยกว่า SX1276 ถึง 50%
เป็น standard สำหรับ Meshtastic และ LoRaWAN ใน 2026 มาพร้อม Heltec, LILYGO T-Beam S3 และ Waveshare LoRa Hat
Wio SX1262 Module ~$5–$9 USD
