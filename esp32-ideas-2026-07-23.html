Wi-Fi CSI ตรวจจับบุคคล — เมื่อ Router กลายเป็น Radar

WaveSight / ESPectre · ESP32 ทุกรุ่น · ไม่ต้องมีกล้อง ไม่ต้องมี mmWave Radar

เทคนิคที่กำลังฮ็อตในชุมชน ESP32 ช่วงนี้คือการใช้ Channel State Information (CSI) จากสัญญาณ Wi-Fi เพื่อตรวจจับการเคลื่อนไหวและการมีอยู่ของมนุษย์ โดยไม่ต้องใช้กล้อง ไมโครโฟน หรือเซ็นเซอร์พิเศษใดๆ เลย ทำให้เกิดประโยชน์ด้าน ความเป็นส่วนตัว สูงมาก

โปรเจกต์ WaveSight (GitHub) และ ESPectre ใช้ ESP32 ดักจับค่า CSI ซึ่งมี 64 ค่าเชิงซ้อน (amplitude + phase) ต่อ subcarrier OFDM เมื่อมีคนเดินผ่าน multipath reflection เปลี่ยนไปเล็กน้อย — RSSI แทบไม่สังเกตเห็น แต่ CSI ตรวจได้ชัดเจน Espressif ยังมี library อย่างเป็นทางการชื่อ esp-csi พร้อม component esp_wifi_sensing

หลักการทำงาน
  1. ESP32 ส่ง Wi-Fi beacon/probe packets ปกติ และตั้งค่าให้เก็บ CSI จาก ACK response
  2. วิเคราะห์ค่า amplitude และ phase ของ 64 subcarriers แต่ละ subcarrier ตลอดเวลา
  3. คำนวณ variance / jitter ในช่วงเวลาสั้น — ถ้าสูงผิดปกติ = มีการเคลื่อนไหว
  4. ส่งสถานะผ่าน MQTT ไปยัง Home Assistant หรือ dashboard Web UI แบบ real-time

ESP32P4_ULP — Arduino Library สำหรับ LP Core ของ ESP32-P4

Xorlent · MIT License · Arduino Library Manager

ESP32-P4 มี LP (Low Power) Core แยกต่างหาก — เป็น RISC-V processor ตัวเต็มที่ทำงานขณะ main CPU หลับอยู่ แต่เดิม Arduino SDK ไม่รองรับ library นี้โดยตรง จนมีคนชื่อ Xorlent ปล่อย library ฟรี MIT license

3 โปรแกรมสำเร็จรูปใน library: (1) gpio_wakeup ตื่นเมื่อ GPIO trigger (2) int_wakeup interrupt-based wakeup (3) soft_i2c_temp_wakeup อ่าน SHT4X sensor แล้วตื่นเมื่ออุณหภูมิ/ความชื้นเกิน threshold — ติดตั้งผ่าน Arduino Library Manager ค้นหา ESP32P4_ULP

Deep Sleep SHT4X Support LP GPIO LP I2C/UART/SPI

Matter 1.4 Certified · DIY Smart Home

ระบบตรวจจับบุคคลไร้กล้อง ด้วย Wi-Fi CSI + ESP32 + Home Assistant

ประยุกต์ใช้ Wi-Fi Channel State Information (CSI) ตรวจจับการมีอยู่ของคนในห้องโดยไม่ต้องพึ่งกล้อง เหมาะสำหรับบ้านที่ต้องการ Home Security แบบ Privacy-First

เชื่อม MQTT กับ Home Assistant เพื่อทริกเกอร์ไฟ แอร์ หรือระบบแจ้งเตือน ความแม่นยำเพิ่มขึ้นด้วย calibration และ machine learning เบื้องต้น

อุปกรณ์ที่ต้องการ ESP32 (ทุกรุ่น ตั้งแต่ ESP32 classic จนถึง ESP32-C6) · Router Wi-Fi ที่ส่ง signal ไปยัง ESP32 · MQTT Broker (Mosquitto) · Home Assistant · library esp-csi หรือ WaveSight (GitHub: ErfanDL/WaveSight)

กล้อง Edge AI Privacy-Safe ด้วย ESP32-P4 + Arducam IMX500

สร้างกล้อง Object Detection ที่ ไม่ส่งภาพออก ออกมาสู่ภายนอก — ประมวลผลทั้งหมดใน IMX500 แล้วส่งแค่ผลลัพธ์ (เช่น "พบคน 2 คน" หรือ "พบรถยนต์") ผ่าน Wi-Fi ไปยัง server

เหมาะสำหรับงาน Worker Safety Monitoring ในโรงงาน, ระบบนับคนใน store ขนาดเล็ก, หรือ Smart Parking ที่ต้องการเก็บ privacy ของผู้ใช้

อุปกรณ์ที่ต้องการ DFRobot FireBeetle 2 ESP32-P4 ($11.90) · Arducam IMX500 MCU AI Camera Module · ESP-IDF v6.x + esp-dl library · MQTT / HTTP endpoint สำหรับรับผลลัพธ์

Wearable Health Monitor ด้วย ESP32-S3 + MAX30102 + DS18B20

สร้างอุปกรณ์ติดตามสุขภาพแบบสวมใส่ได้ วัด Heart Rate, SpO2, และอุณหภูมิร่างกาย แบบ real-time ส่งข้อมูลผ่าน BLE ไปยัง smartphone app หรือ Web Dashboard

งานวิจัย 2026 ชี้ให้เห็นว่า ESP32-S3 มี deep sleep ที่ประหยัดพลังงานพอสำหรับใส่แบตเตอรี่ LiPo 400 mAh ได้นานหลายวัน โดยเฉพาะเมื่อส่งข้อมูลผ่าน BLE (ไม่ใช่ Wi-Fi)

อุปกรณ์ที่ต้องการ XIAO ESP32-S3 (ขนาดเล็กเหมาะสวมใส่) · MAX30102 (Heart Rate + SpO2) · DS18B20 (Body Temperature) · LiPo 400mAh + TP4056 charger · OLED 0.96" SSD1306 สำหรับแสดงผล

Ultra Low Power Field Sensor ด้วย ESP32-P4 LP Core + SHT45

ใช้ ESP32P4_ULP library ใหม่ให้ LP core อ่านค่า SHT45 sensor (อุณหภูมิ/ความชื้น) ทุก 5 นาทีในขณะที่ main CPU หลับอยู่ ตื่นเฉพาะเมื่อค่าเกิน threshold ที่กำหนด

เหมาะสำหรับ โรงเรือนเพาะปลูก, ห้องเก็บของ, หรือ Cold Chain Monitoring ที่ต้องการแบตเตอรี่อยู่ได้นานเป็นเดือน ไม่ต้องเปลี่ยนบ่อย

อุปกรณ์ที่ต้องการ DFRobot FireBeetle 2 ESP32-P4 · SHT45 (I2C, ความแม่นยำ ±0.1°C) · LiPo 1000mAh · Arduino IDE + ESP32P4_ULP library (Arduino Library Manager) · MQTT broker สำหรับ data logging

RTD Sensor (Pt100/Pt1000) — อุณหภูมิ Industrial Grade

4-Wire · ±0.01°C · -200 ถึง +850°C

RTD (Resistance Temperature Detector) แบบ 4-Wire ขจัดความผิดพลาดจากความต้านทานสาย ให้ความแม่นยำ ±0.01°C เหมาะกับงาน industrial, ห้องปฏิบัติการ, HVAC monitoring

ต่อกับ ESP32 ผ่าน MAX31865 module (SPI) ราคาประมาณ 120 บาท ใช้ร่วมกับ Pt100 probe ได้ทันที มี Arduino/IDF library พร้อมใช้
