WakeNet7 + ESP-SR — Wake Word Detection <80ms บน ESP32-S3

Espressif ESP-SR library เวอร์ชันล่าสุดรองรับ WakeNet7 ตรวจจับ "Hi ESP" ในเวลา <80ms หลัง utterance จบ ซึ่งต่ำกว่า threshold การรับรู้ของมนุษย์ที่ 300ms และรองรับ MultiNet สำหรับ offline speech command ได้ถึง 200 คำสั่ง

Latency
< 80ms wake word detection
Wake words
"Hi ESP", "Hi Lexin" + custom
Commands

Hardware
ESP32-S3 + I²S Microphone (e.g. INMP441)

Hardware

Library

Protocol
Modbus RTU → MQTT → Node-RED
Extra
Watchdog + FOTA Update + Deep Sleep
⚠️ ต้องใส่ termination resistor 120Ω ที่ปลายทั้งสองข้างของ bus RS-485 เพื่อป้องกัน signal reflection โดยเฉพาะเมื่อใช้สาย >10m หรือ baud rate สูง

🌬️ All-in-One Environmental
Sensirion SEN66
เซนเซอร์สิ่งแวดล้อมครบวงจรในโมดูลเดียว รวม PM sensor รุ่นใหม่ MEMS-based (SPS6x) + CO₂ + VOC + NOx วัดได้ถึง 9 ค่า ผ่าน I²C เดียว ราคา ~$25 แทนที่เซนเซอร์ 3 ตัวได้เลย
PM1 / PM2.5 / PM4 / PM10 CO₂ (NDIR) VOC Index NOx Index Temperature Humidity I²C interface
Project Aura 2026 ใช้ SEN66 + ESP32-S3 + หน้าจอ 4.3" IPS touchscreen เป็น all-in-one air quality monitor ที่ต่อ Home Assistant ผ่าน MQTT Auto-Discovery ได้เลย
📳 Vibration / IMU
ADXL355 (Analog Devices)
Accelerometer ความละเอียดสูง 20-bit สำหรับงาน industrial vibration analysis — noise ต่ำกว่า ADXL345 มาก เหมาะกับ FFT spectrum analysis ตรวจจับ bearing fault และ imbalance ในมอเตอร์
±2g / ±4g / ±8g 20-bit ADC SPI / I²C ODR 4000 SPS Noise: 25 μg/√Hz Low drift (temp.)
ดีกว่า MPU-6050 / ADXL345 สำหรับงาน predictive maintenance เพราะ noise floor ต่ำกว่า 10x — ตรวจจับ micro-vibration ที่ล้อลูกปืนสึกหรอได้ตั้งแต่เริ่มต้น
📡 LoRa Transceiver
Semtech SX1262 LoRa Module
LoRa transceiver รุ่นล่าสุด รองรับ LoRaWAN 1.0.4 / 1.1 ระยะส่ง 15 km (LOS) กำลังส่งสูงสุด +22 dBm รองรับ Class A/B/C สิ้นเปลือง current ขณะรับ 4.6 mA เท่านั้น เหมาะมากสำหรับ node พลังงานต่ำ
Freq: 868/915 MHz +22 dBm TX -148 dBm sensitivity 4.6 mA RX LoRaWAN 1.1 SPI interface
🎤 Microphone
INMP441 (Knowles I²S MEMS Mic)
ไมโครโฟน MEMS digital ต่อกับ ESP32-S3 ผ่าน I²S โดยตรง — คุณภาพเสียงดีมาก SNR 61 dB เหมาะสำหรับ wake word detection และ speech recognition ด้วย WakeNet7 / microWakeWord
I²S Digital output SNR: 61 dB(A) Freq: 60 Hz – 15 kHz Supply: 1.8–3.3V Omnidirectional
เชื่อมต่อ ESP32-S3 ด้วย I²S เพียง 4 สาย (SCK, WS, SD, GND) ใช้งานกับ esp-skainet หรือ ESPHome voice_assistant component ได้ทันที
🏭 Industrial Interface
ADM3485 / MAX485 RS-485 Transceiver
IC แปลง TTL UART จาก ESP32 เป็นสัญญาณ differential RS-485 สำหรับการสื่อสาร Modbus RTU ในสภาพแวดล้อมอุตสาหกรรม ทนทานต่อ noise, รองรับสายยาว 1,200 เมตร, ต่อ device ได้ถึง 32 node บน bus เดียว
Bus length: 1,200 m Speed: 10 Mbps 32 devices/bus ±15 kV ESD 3.3V / 5V Half-duplex
ใส่ termination 120Ω ที่ปลาย bus เสมอ และใช้ RC filter (100Ω + 100pF) ที่ A/B line เพื่อกรอง spike จากสภาพแวดล้อมโรงงาน
