ระบบตรวจจับศัตรูพืชด้วย AI
ESP32-P4 + กล้อง + TFLite Micro
ติดตั้ง ESP32-P4 พร้อมกล้องบนโดรนหรือแขนกล วิ่ง TFLite Micro โมเดลที่ฝึกบน dataset ใบโรค แมลง หนอน ตรวจจับศัตรูพืชแบบ real-time พร้อมส่ง JPEG + แจ้งเตือนผ่าน ESP32-C6 (ทำงานเป็น Wi-Fi companion)
  • Inference time ~200ms ที่ 96×96 input
  • ทริกเกอร์ precision sprayer relay อัตโนมัติ
  • ประหยัดสารกำจัดศัตรูพืชได้ถึง 60%

วิทยุสื่อสาร ESP-NOW
2 บอร์ด ESP32 + ไมค์ + ลำโพง
สร้างวิทยุสื่อสาร (Walkie-Talkie) จาก ESP32 2 ตัว โดยใช้ ESP-NOW protocol ความหน่วงต่ำ ไม่ต้องการ Wi-Fi router หรืออินเทอร์เน็ต สื่อสารแบบ peer-to-peer ได้เลย
  • ESP-NOW: latency < 10ms
  • ระยะทาง outdoor ~480 เมตร
  • ไม่ต้องมี infrastructure ใดเลย

Project Aura — Air Quality Monitor
ESP32-S3 + Sensirion + 4.3" IPS Touchscreen

  • วัด PM2.5, CO2, VOCs, อุณหภูมิ, ความชื้น
  • Native Home Assistant integration
  • Open source hardware + firmware

Edge AI Security System

  • Person detection ด้วย MobileNetV1 ~200ms
  • Wake word detection แบบ local
  • Privacy-first: ประมวลผลในตัว 100%

  • เหมาะกับ prototype rapid ที่ต้องการ compute สูง

Sensors ที่น่าสนใจสำหรับ ESP32
Sensors ใหม่และการประยุกต์ใช้งานปี 2026
🌟 EnviroGo (2026) — Wearable สำหรับ ESP32-S3 ที่บรรจุ 7 sensors ในตัวเดียว: VOCs, UV index, อุณหภูมิ, ความชื้น, air pressure, light intensity, และ 9-axis MEMS motion — ใช้แบตเตอรี่ สวมใส่ได้ หรือติดกระเป๋า ดีไซน์สำหรับ indoor/outdoor
01

Sensirion SEN55 (PM + VOC + NOx)

ระดับ industrial วัด PM1/2.5/4/10, VOC index, NOx index, อุณหภูมิ และความชื้น ใช้ใน Project Aura

02

ICM-42688 — 9-Axis MEMS IMU

Accelerometer + Gyroscope + Magnetometer ความแม่นยำสูง สำหรับ gesture recognition, fall detection, wearable

03

MQ series Gas Sensors (CO/CH4/LPG)

ตรวจจับ Carbon Monoxide, Methane, LPG เชื่อม Blynk หรือ Home Assistant แจ้งเตือนความปลอดภัย

04

VEML6075 UV Index Sensor

วัด UVA และ UVB radiation แบบ digital I2C เหมาะสำหรับ wearable outdoor monitoring

05

Atlas Scientific Water Quality

pH, EC, DO, ORP สำหรับระบบน้ำ hydroponics หรือ aquaculture monitoring ด้วย ESP32

06

OV5640 / OV2640 Camera Module

กล้องสำหรับ ESP32-P4 รองรับ MIPI-CSI hardware ISP ถ่าย JPEG ส่งผ่าน companion C6

07

SCD40/SCD41 CO2 Sensor (Sensirion)

วัด CO2 แบบ photoacoustic ขนาดเล็ก ความแม่นยำสูง รองรับ I2C ใช้พลังงานต่ำ

08

MLX90640 IR Array Camera

กล้องความร้อน 32×24 pixel ราคาประหยัด สำหรับ thermal imaging, presence detection บน ESP32-S3

09

MAX30102 Pulse Oximeter

วัด SpO2 และ Heart Rate ใช้กับ wearable health monitor บน ESP32-S3 ส่งข้อมูลผ่าน BLE

10

BME688 AI-Enabled Gas Sensor (Bosch)

วัด VOC, humidity, pressure, temperature พร้อม AI บนชิปสำหรับ gas pattern recognition
