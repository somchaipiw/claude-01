ESP32-C5 — Dual-Band Wi-Fi 6 (2.4 + 5 GHz)
ชิปตัวแรกในตระกูล ESP32 ที่รองรับ Wi-Fi 6 ทั้ง 2.4 GHz และ 5 GHz พร้อมกัน ทำให้สามารถเลือกย่านความถี่ที่ไม่แออัดได้ และยังรองรับ Bluetooth 5 LE, Zigbee 3.0 และ Thread 1.3 ด้วย
CPURISC-V single-core @ 240 MHz
Wi-Fi802.11ax (Wi-Fi 6) — 2.4 + 5 GHz
BluetoothBLE 5.0
ProtocolZigbee 3.0, Thread 1.3, Matter
RAM384 KB SRAM + 8 MB PSRAM
Flash16 MB / 32 MB
ทำไมต้อง 5 GHz? ในสภาพแวดล้อมที่มี Wi-Fi หลายเครือข่าย ย่าน 5 GHz มีความแออัดน้อยกว่า ทำให้ latency ต่ำลงและ throughput สูงขึ้น เหมาะสำหรับกล้อง IP, สตรีมมิ่ง, และโปรเจกต์ที่ต้องการแบนด์วิดท์สูง

ESP32-C61 — Budget Wi-Fi 6 + Secure TEE
ชิปที่ออกแบบให้ราคาถูกที่สุดในกลุ่มที่รองรับ Wi-Fi 6 เหมาะสำหรับโปรเจกต์ที่ต้องการ wireless ปลอดภัยในราคาประหยัด มี Trusted Execution Environment (TEE) ป้องกัน private key ใน hardware
CPURISC-V single-core @ 160 MHz
Wi-Fi802.11ax (2.4 GHz เท่านั้น)
BluetoothBLE 5.0
RAM320 KB SRAM + in-package PSRAM
SecuritySecure Boot + Flash Encryption + TEE
ECDSAHardware Digital Signature Peripheral
Use case: Co-processor สำหรับบอร์ดที่ยังไม่มี wireless หรือใช้เป็น MCU หลักในโปรเจกต์ IoT ที่ต้องการความปลอดภัยสูงในราคาต่ำ

FRAMEWORK UPDATE
ESP-IDF v6.0 — AI Integration, PSA Crypto & OTA Safety
Framework หลักสำหรับ ESP32 ออกเวอร์ชัน major ใหม่ในเดือนมีนาคม 2026 มาพร้อม breaking changes และฟีเจอร์ที่ developer ต้องรู้
1
MCP Server Built-in — รองรับ AI development tools (VS Code Copilot, Cursor) ให้ทำงานกับ ESP32 project ได้โดยตรง ผ่าน Model Context Protocol
2
Picolibc แทน Newlib — C library ใหม่ที่เบาและเร็วกว่า ลด memory footprint ของ firmware ลงได้มาก
3
PSA Crypto API + MbedTLS 4.x — มาตรฐาน cryptographic ใหม่ที่ปลอดภัยและบำรุงรักษาง่ายขึ้น
4
Recovery Bootloader OTA — อัปเดต firmware ปลอดภัยขึ้น มีพาร์ติชัน backup ป้องกันบอร์ดเสียหายถ้าไฟดับกลางคัน
5
Wi-Fi USD + WPA3 Mode — Proximity-based service discovery และ compatibility mode ใหม่สำหรับ WPA3
