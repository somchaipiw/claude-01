NEW · Chip
ESP32-S31 — Dual-Core RISC-V พร้อม Bluetooth LE Audio
ชิปใหม่จาก Espressif ESP32-S31 เป็น dual-core RISC-V รองรับ Wi-Fi 6 (802.11ax) + Bluetooth 5.4 ที่มี LE Audio และ Classic BR/EDR ในชิปเดียว เหมาะกับการสร้างลำโพงอัจฉริยะ, Voice-Controlled IoT, Smart Speaker และ Hearing Aid รุ่นใหม่ ที่ต้องการคุณภาพเสียง Hi-Fi พร้อม latency ต่ำแบบ Isochronous
🎵 Bluetooth LE Audio ใน ESP32-S31 รองรับ LC3 codec ที่ให้คุณภาพเสียงสูงกว่า SBC/AAC ที่ bitrate ต่ำกว่า — สำหรับ hearing aids และ TWS earbuds รุ่นต่อไป
DIY · Voice Comm

บอร์ด LILYGO T3-S3 MVSR รวม ESP32-S3, transceiver SX1262/SX1280, ไมโครโฟน และลำโพงในบอร์ดเดียว พร้อม sketch Walkie-Talkie ใช้ Codec2 3200 bps ส่งเสียงผ่าน LoRa FSK ที่ 914.6 MHz ได้ระยะ 2–5 กม. ในเมือง เหมาะสำหรับการสื่อสารฉุกเฉิน กิจกรรมกลางแจ้ง หรือ Off-Grid communication network แบบ P2P

Sensors & Modules น่าสนใจ

ชิปและ Module ที่ใช้กับโปรเจกต์วันนี้ — Audio, Automotive, City, Robotics

🔊
TAS5825M
I²S Class-D DAC + Amplifier
DAC พร้อม Amplifier ในชิปเดียวจาก Texas Instruments กำลังสูงสุด 32W/4Ω ต่อ channel รับ I²S input โดยตรงจาก ESP32 ควบคุมผ่าน I²C ประหยัดพลังงาน Class-D THD+N น้อยกว่า 0.03% เหมาะกับ Smart Speaker และ Hi-Fi IoT
📻
SX1262 / SX1280
LoRa Transceiver (Sub-GHz / 2.4 GHz)
Semtech SX1262 สำหรับ Sub-GHz LoRa (433/868/915 MHz) และ SX1280 สำหรับ 2.4 GHz LoRa ใช้ FSK mode ส่ง Audio Codec2 Bitstream ได้ระยะ 2–10 กม. ใน LOS เหมาะกับ Walkie-Talkie, Emergency Mesh, Off-Grid Voice Network
🎯
Benewake TF-Luna
Micro LiDAR ToF Sensor
LiDAR ขนาดเล็ก (35×35×21 mm) วัดระยะ 0.2–8 m ความเร็ว 250 Hz Interface: UART หรือ I²C กินกระแส 80 mA ที่ 5V เหมาะกับ SLAM Robot 360° Scanner และ Obstacle Detection ราคา ~$20–30
🚗
SN65HVD230
CAN Bus Transceiver (3.3V)
CAN Transceiver 3.3V จาก TI เชื่อมต่อ ESP32 TWAI Controller กับ CAN Bus ของรถได้ทันที รองรับ CAN 2.0A/B, ISO-11898 ป้องกัน ESD ±12kV ทนทาน ใช้ในโปรเจกต์ OBD2 Logger, J1939 Truck Monitor ราคา ~$1–3
📡
Quectel BG95-M3
LTE-M / NB-IoT / EGPRS Modem
Modem IoT แบบ LPWA รองรับ LTE-M, NB-IoT และ GSM Fallback ใช้ใน ESPNetLink เชื่อม WiCAN Pro ส่ง Vehicle Telematics ผ่าน 4G ได้ทุกที่ มี Built-in GNSS สำหรับ GPS Tracking ในตัว ขนาด LCC 26×26 mm
🔊
INMP441 / MSM261S4030H0
MEMS I²S Microphone
Omnidirectional MEMS Mic ส่งออกเป็น I²S ดิจิทัลโดยตรงสู่ ESP32 SNR 61 dBA, Sensitivity -26 dBFS AOP 120 dBSPL เหมาะกับ Smart Speaker Voice Assistant และ Audio IoT ราคาถูก ~$1–3 ต่อตัว
